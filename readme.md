# This is a project where I use google authenticaion

## There are two main routes in this project

1. '/auth/login'
2. '/user/profile'

### steps to start the project

1. for install necessary libraries

```console
$ npm install
```

2. start the project

```console
$ npm run dev
```

### Packages

1. express
2. mongoose
3. passport
4. passport-google-oauth20
5. express-session
6. dotenv
7. ejs
